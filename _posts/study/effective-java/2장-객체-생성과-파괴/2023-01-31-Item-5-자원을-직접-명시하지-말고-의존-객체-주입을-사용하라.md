## 아이템 5. 자원을 직접 명시하지 말고 의존 객체 주입을 사용하라

### 의존객체 주입하기

어디선가 많이 본 듯한 주입 방법

```java
public class Hello {
  private final World world;
  
  public Hello (World world) {
    this.world = world;
  }
}
```

##### 장점

이렇게 의존객체를 주입해서 사용하는 것은 유연성과 테스트 용이성을 높인다.
